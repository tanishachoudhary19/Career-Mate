What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It tells the browser how to display text, images, and other elements. HTML is the backbone of all websites and is the first step to becoming a web developer.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first HTML webpage.</p>
</body>
</html>

HTML Elements

HTML documents are made up of elements that usually have an opening tag, content, and a closing tag.

  • <h1>...</h1> - Headings
  • <p>...</p> - Paragraph
  • <a href="url">...</a> - Links
  • <img src="image.jpg"> - Images

Headings in HTML

Headings are defined with <h1> to <h6>.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Heading</h3>

Links in HTML

Links allow users to navigate from one page to another.

<a href="https://www.example.com">Visit Example</a>

Images in HTML

Images are added using the <img> tag.

<img src="photo.jpg" alt="Description">

Tables in HTML

Tables are used to display data in rows and columns.

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

Forms in HTML

Forms allow users to input data.

<form>
    <input type="text" placeholder="Enter Name"><br>
    <input type="email" placeholder="Enter Email"><br>
    <button type="submit">Submit</button>
</form>

Semantic HTML

HTML5 introduced semantic elements that clearly describe their meaning.

  • <header> - Top section
  • <nav> - Navigation menu
  • <article> - Article content
  • <footer> - Footer section
