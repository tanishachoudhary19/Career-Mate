Core AWS Services

  • EC2 (Elastic Compute Cloud): Virtual servers in the cloud.
  • S3 (Simple Storage Service): Secure and scalable storage.
  • RDS (Relational Database Service): Managed databases.
  • Lambda: Serverless computing service.

Example: Hosting a Website on AWS S3

1. Create an S3 bucket in your AWS account.
2. Upload your website files (HTML, CSS, JS).
3. Enable static website hosting in the bucket properties.
4. Make files public and use the provided S3 website URL.

AWS Pricing Model

AWS follows a pay-as-you-go pricing model, meaning you only pay for the services you use. This allows flexibility for startups and enterprises alike.
