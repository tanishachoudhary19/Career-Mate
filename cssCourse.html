Front End Development - CSS

Style your web pages beautifully with CSS

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages â€” for example, to change fonts, colors, spacing, and positioning of HTML elements. CSS makes your website look attractive and user-friendly.

How CSS Works

CSS works by selecting HTML elements and applying styles to them. These styles are applied using selectors and properties.

Types of CSS

  • Inline CSS - Styles applied directly inside the HTML tag.
  • Internal CSS - Styles written inside a <style> tag in the HTML file.
  • External CSS - Styles written in a separate .css file and linked to the HTML file.
/* Example External CSS */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}
h1 {
    color: royalblue;
    text-align: center;
}

Selectors in CSS

  • p - Selects all paragraph elements.
  • .classname - Selects elements with a specific class.
  • #idname - Selects an element with a specific ID.
  • * - Selects all elements.

Colors and Backgrounds

body {
    background-color: lightblue;
}
p {
    color: darkgreen;
}
div {
    background-image: url("background.jpg");
}

Text Styling

h1 {
    font-size: 40px;
    font-weight: bold;
    text-align: center;
}
p {
    line-height: 1.6;
    font-style: italic;
}

Box Model

The CSS box model describes how the size of an element is calculated: content, padding, border, margin.

div {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

CSS Layout Techniques

  • Flexbox - Flexible layouts.
  • Grid - Two-dimensional layouts.
  • Positioning - Absolute, relative, fixed, sticky.
