What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It’s widely used in web development, data science, artificial intelligence, automation, and more.

Why Learn Python?

  • Easy to learn and beginner-friendly.
  • Versatile — used in web, AI, data science, and automation.
  • Huge community support.
  • High demand in job market.

Python Basics

Here are some fundamental concepts:

  • Variables: Store data values.
  • Data Types: String, Integer, Float, Boolean, List, Tuple, Dictionary.
  • Operators: +, -, *, /, %, =, ==, etc.
  • Functions: Reusable blocks of code.
name = "John"  # String
age = 25       # Integer
is_student = True  # Boolean

def greet():
    print("Hello, " + name)

greet()  # Output: Hello, John

Control Structures

    print("Grade: A")
elif score >= 75:

else:
